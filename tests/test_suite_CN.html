Pocketsphinx.js test suite for Chinese models

This test suite is based on the Chinese acoustic and language models provided with PocketSphinx (the original, from CMU). To get it working, you need to package the model files with the following cmake command:

cmake .. -DCMAKE_TOOLCHAIN_FILE=path_to_emscripten/cmake/Platform/Emscripten.cmake -DHMM_BASE=path_to_pocketsphinx/model/hmm/zh -DHMM_FOLDERS=tdt_sc_8k -DDICT_BASE=path_to_pocketsphinx/model/lm/zh_CN -DDICT_FILES=mandarin_notone.dic -DLM_BASE=path_to_pocketsphinx/model/lm/zh_CN/ -DLM_FILES=gigatdt.5000.DMP

You can then build it, and rename pocketsphinx.js into pocketsphinx_CN.js
