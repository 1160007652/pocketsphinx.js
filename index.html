Welcome to Pocketsphinx.js.

This project is an attempt to create a speech recognition library entirely in JavaScript. The idea is to make use of Emscripten to convert PocketSphinx into JavaScript and feed the recognizer with audio recorded with the Web Audio API.

Current status.

It pretty much works, assuming you are using Chrome with a functional audio hardware setting. We have found that on many platforms and hardware, the audio data recoded through the Web Audio API are silent, this will hopefully be addressed soon.

Live demo.

Try it out there. This will load pocketsphinx.js which is very large (15MB), so be patient.
